# GraphQL Playlist

To help learn GraphQL.
Alongside React, Node.js, Express.js, MongoDB.

## Getting Started

### Requirements

Node.js

### Installation

1. `cd server`
2. `npm start`

### Testing

Graphiql is tied to /graphql path. Visit http://localhost:4000/graphql after local app started and run queries using the UI at this url.
